#天气查询应用 开发指令

## 项目概述
这是一个简单天气查询应用，前端使用 React + TypeScript，天气查询API调用 OpenWeatherMap API

## 开发规范
- 使用函数式组件 + Hooks
- 使用 Tailwind CSS 编写样式
- 组件要可复用
- 代码要有注释-中文注释
- 请使用中文来写文档

## 代码风格
- 使用 ESLint 和 Prettier
- 组件名使用 PascalCase
- 函数名使用 camelCase
- 常量使用 UPPER_SNAKE_CASE

## 设计要求
- 明亮主题，简约色调
- 添加平滑移动的滚动动画
- 确保移动端适配

## 测试要求
- 每个功能完成后手动测试
- 确保数据正确存储和读取
- 测试各种边界情况

## 注意事项
- 保持代码简洁，避免过度设计
- 优先实现核心功能
- 确保移动端适配

## 代码提交
- 每次完成有意义的代码或文件修改后，使用 git commit 指令提交修改
- 自主修改 .gitignore 文件，项目中依赖文件或日志文件或测试功能等代码或环境变量文件无需提交
- 提交命名请遵照：feat/chore/docs/fix/others: 中文描述
