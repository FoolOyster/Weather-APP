# 天气查询应用 PRD

## 产品概述
一个简单的天气查询应用，提供便捷的天气查询功能，调用OpenWeatherMap API

## 目标用户
随手点开网站想快速查询天气的用户（主要还是用于开发者个人）

## 核心功能

### 必须做（MVP）
1. 天气查询功能
    - 用户输入城市查询当天天气
    - 显示天气状况、温度、湿度、风速等信息
    - 温度显示当前温度，当天最高温和最低温

2. 显示未来天气预报
    - 查看当前城市未来十天的天气情况
    - 未来十天的最高温和最低温

3. 收藏城市功能
    - 可收藏查询成功过的城市
    - 方便下次查询

### 后续可以做
- 可根据当天天气状况调整当前背景主色调，如晴天时色调为天空蓝，阴天或雨天时色调变灰暗
- 可添加天气图片和动画，让界面变得更生动
- 显示空气质量，依据AQI优劣程度提出当天合适建议，如“适宜户外运动”等其他提示语
- 添加一个当天天气状况的详细数据界面，内容包含当天紫外线、日落日出时间、风速、湿度露点、能见度、体感温度
- 支持多城市对比，一次查看多个城市的天气
- 添加天气预警，及时提醒恶劣天气
- 可查询历史天气，分析天气趋势

## 界面设计
- 主要为明亮主题，天空蓝色调
- 可用emoji表示相关天气状况，用条形图表示当天最高温和最低温
- 简洁布局
- 响应式设计，移动端可用
